SimpleExplosionStrategyexplosionRangePositionsGroupedByPower Field
The explosion range positions grouped by power. The outer List has 5 elements (lists), representing the 5 possible explosive powers. The middle List has all the relative cell positions for a specific explosive power (only the unique positions) A relative position is represented by an int array of 2 elements - row and column, witch are to be added to the detonation position in order to get the coordinates of the specified position.
